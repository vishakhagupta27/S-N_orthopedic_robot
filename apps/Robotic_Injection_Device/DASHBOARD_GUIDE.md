
## 🆘 Troubleshooting

### Logo Not Displaying
**Issue**: Capgemini logo not showing
**Fix**: Ensure `Cap_logo.png` is in the same directory
```bash
ls -la Cap_logo.png
# Should show the file exists
```

### CSS Not Loading
**Issue**: Styling appears broken
**Fix**: Ensure you're opening the file via a web server or `file://` protocol

### Auto-Refresh Not Working
**Issue**: Test results dashboard not refreshing
**Fix**: Check browser console for JavaScript errors
